# 📸 Image Gallery Project

A responsive image gallery built using HTML, CSS, and JavaScript.

## ✨ Features
- Lightbox preview
- Next/Previous navigation
- Fade animation
- Swipe support (mobile)
- Thumbnail preview
- Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
